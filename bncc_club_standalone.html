การเชื่อมโยง Google Sheets / Cloud API

ระบบสถาปัตยกรรมทำงานบนคลาวด์โดยมี Google Sheets เป็นฐานข้อมูลหลัก และประมวลผลผ่าน Apps Script คุณสามารถคัดลอกไฟล์สำรองข้อมูลดิบ (JSON Database) ไปทดสอบหรืออัปเดตเข้าระบบฐานข้อมูลคลาวด์ได้ที่นี่

สรุปขั้นตอนสำหรับผู้พัฒนาระบบ (Developer Info)

ในระบบการใช้งานจริง คุณสามารถผูกฟังก์ชัน JavaScript เหล่านี้ไปยัง Google Apps Script (GAS) โดยใช้ Web App API:

  1. นำเข้าไลบรารีสคริปต์นี้ไปยังโครงการ Google Apps Script
  2. เขียนฟังก์ชัน doGet(e) และ doPost(e) เพื่อเปิด API endpoint บน Google Cloud
  3. เชื่อมโยงตารางข้อมูลของ Google Sheets กับ JSON ในเครื่องของเบราว์เซอร์ โดยการเขียนคำสั่งอ่าน-เขียนข้อมูลเซลล์ของชีต เช่น SpreadsheetApp.getActiveSpreadsheet().getSheetByName("members")
  4. ระบบทะเบียนสมาชิกและสวัสดิการนี้จะส่งคำขอ fetch ไปยังปลายทาง GAS เพื่อให้ผู้ใช้งานทุกคนเข้าข้อมูลพร้อมกันเรียลไทม์
ข้อดีของการทำงานบน Cloud (Serverless):
ไม่ต้องจัดเตรียมเซิร์ฟเวอร์, ประหยัดค่าใช้จ่าย, มีประวัติการทำงานย้อนหลังผ่าน Google Drive, และบำรุงรักษาโดยคณะกรรมการได้ง่ายด้วยตนเอง
